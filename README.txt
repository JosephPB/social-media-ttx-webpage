# Social Post Exercise Website

Files:
- index.html
- styles.css
- script.js

Features included:
- 3-column masonry-style desktop layout
- Facebook and X post cards
- training badge in top-right
- downloadable attached media
- profile card flip interaction
- editable profile metadata in the JavaScript array

How to use:
1. Replace placeholder files in:
   - assets/profiles/
   - assets/media/
2. Update the posts array in script.js
3. Open index.html in a browser

If your browser blocks some local media features when opening files directly,
run a small local server instead, for example:

python3 -m http.server 8000

Then open:
http://localhost:8000
